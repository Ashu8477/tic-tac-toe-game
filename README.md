# 🎮 Tic Tac Toe Masters

A modern and responsive Tic Tac Toe game built using **HTML, CSS, and JavaScript**. Play against your friend or challenge an AI with multiple difficulty levels, smooth animations, sound effects, and a clean user interface.

## 🌐 Live Demo

👉 https://tic-tac-toe-masters.netlify.app/

---

## ✨ Features

- 🎮 Player vs Player Mode
- 🤖 Player vs AI Mode
- 🟢 Easy AI
- 🟡 Medium AI
- 🔴 Hard AI
- 🏆 Winner Detection
- 🤝 Draw Detection
- 🔄 Restart Game
- 📊 Score Board
- 🔊 Sound Effects
- ✨ Winning Line Highlight
- 📱 Fully Responsive Design
- ⚡ Smooth UI & Animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6 Modules)

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Ashu8477/tic-tac-toe-game.git
```

Open the project

```bash
cd tic-tac-toe-game
```

Run locally

Simply open **index.html** in your browser.
